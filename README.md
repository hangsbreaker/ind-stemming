# ind-stemming-js
Javascript Stemming Bahasa Indonesia

# Usage for nodejs
const { stem } = require("./stem");<br>
console.log(stem("Kalimat percobaan ini telah berubah menjadi kata dasar"));
<br><br>
// result<br>
// kalimat coba ini telah ubah jadi kata dasar
